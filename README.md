## DevTools Trainer (без бэкенда)

Статическое веб-приложение для тренировки навыков работы с DevTools.

## Что внутри

На стартовом экране есть выбор режима:

- **Тренажер 1 уровень** — базовые 20 заданий с подсказками.
- **Тренажер 2 уровень** — 20 инцидентных задач без пошаговых подсказок.

Покрываемые инструменты:

- Elements
- Console
- Network
- Application (LocalStorage, SessionStorage, Cookies, IndexedDB)

## Запуск

Важно: для корректной работы `fetch` и Service Worker открывайте через локальный сервер, не через `file://`.

### Вариант 1 (Python)

```bash
python -m http.server 5173
```

Откройте в браузере:

- `http://localhost:5173` (рекомендуется)
- `http://127.0.0.1:5173`

### Вариант 2 (Node)

```bash
npx serve -l 5173 .
```

## Важно про Service Worker

Для Network-задач приложение использует `sw.js`.

Если вы открыли сайт на `0.0.0.0`, Service Worker может не активироваться в браузере, и часть задач (особенно с response headers) будет работать некорректно.

Если это произошло:

1. Откройте сайт через `localhost`.
2. Сделайте hard reload.
3. Проверьте, что внизу отображается `Service Worker: активен`.

## Прогресс

Прогресс хранится отдельно для каждого режима в `localStorage`.

Сброс:

- кнопка **Сбросить прогресс** в интерфейсе,
- или ручная очистка `localStorage`.
